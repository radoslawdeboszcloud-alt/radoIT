# RadoIT – Single Page (GitHub Pages)

To jest prosta, statyczna strona (HTML/CSS/JS) z przełącznikiem języka DE/PL.
Nie wymaga Node.js ani builda.

## Struktura
- `index.html`
- `styles.css`
- `script.js`
- `assets/logo.png`

## Uruchom lokalnie
Otwórz `index.html` w przeglądarce (lub użyj Live Server w VS Code).

## GitHub Pages
Instrukcja publikacji jest w odpowiedzi ChatGPT (krok po kroku).
